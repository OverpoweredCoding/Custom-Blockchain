## Custom-Blockchain
A custom blockchain (feel free to contribute)

## Installation (js)
```
$ npm i

$ node index.js
--OR--
$ node .
```

## Installation (pip)
```
$ pip install pipenv
$ pipenv install

$ pipenv run python blockchain.py
--OR--
$ pipenv run python blockchain.py -p 5001
--OR--
$ pipenv run python blockchain.py --port 5002
```
